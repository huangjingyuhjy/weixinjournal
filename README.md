一个日志提醒小程序
基于微信云坏境开发
包含用户登录，添加修改日志，打卡，商城等功能
需要配置相应云环境开发
包含登录功能，签到功能，添加修改功能，提醒功能，积分兑换功能等功能，增加了支付购买积分的界面，暂未实现相应功能
用户登录界面，实现登录小程序，使用微信账号登录，即直接使用本机微信账号作为用户，完成登录操作，或直接登出，退出小程序，同时使用openid获取用户微信账号的基本信息
在签到页面实现实现签到功能，登录后自动弹出，每日签到一次，签到完成后获得任务奖励，显示连续签到天数，以及退出小程序功能。
主页面有添加修改按钮，我的界面按钮，日志内容显示，主界面按钮，完成小程序的主要功能
我的页面包括个人信息展示栏，切换用户按钮，积分商城按钮，反馈客服按钮，任务完成情况按钮
